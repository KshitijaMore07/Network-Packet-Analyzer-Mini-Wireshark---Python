# 🌐 Network Packet Analyzer (Mini Wireshark)

A Python-based desktop application that captures, analyzes, and visualizes network packets in real time. Built using **Scapy + Tkinter**, this project helps understand how data flows across a network.

---

## 🚀 Features

- 📡 Live packet capture using Scapy  
- 🖥️ User-friendly GUI (Tkinter)  
- 🌍 Displays Source IP, Destination IP, Protocol, Packet Size  
- 🔍 Search packets by IP address  
- 💾 Save & Load capture sessions (JSON)  
- 📊 Export packet data to CSV  
- 📈 Protocol distribution pie chart (Matplotlib)  
- ⏯️ Start / Stop packet capturing  

---

## 🛠️ Tech Stack

- **Language:** Python  
- **GUI:** Tkinter  
- **Packet Capture:** Scapy  
- **Data Visualization:** Matplotlib  
- **Multithreading:** Threading  
- **Data Handling:** JSON, CSV  
- **Utilities:** Queue, datetime  


---

## ⚙️ How It Works

```

Start Application
↓

↓

↓

↓
User can:

- Export CSV

↓
Stop Capture
